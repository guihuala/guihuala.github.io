# The bucket list

在萌芽gj群里看到策划在捞人参加cusga，于是就找上了。然后可以说是，真正意义上的命运的齿轮开始转动...

## demo

也就是参加cusga的demo版本。

开发周期是从25年2月底左右到5月20号，期间因为我同时做了三个坑耽误了一段时间（并非自恋但是真的有一种要是自己不工作，这个项目就几乎停摆了的感觉，尽管项目最终取得成就受益者也不是自己），最后还是带着一堆bug提交了。

本来其实对这个游戏充满失望了，甚至想着cusga结果出来这游戏几乎也不会有什么反响吧，到时候找个借口退出团队好了。

结果也不知道怎么搞的cusga的评委眼光就是很差，还真让这个全是bug和玩法问题剧情问题的demo通过了初赛。别说还真是因为这个入围初赛让我舍不得退出了，总想着还是要做完。
